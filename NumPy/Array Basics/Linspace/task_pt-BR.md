## Linspace

### Tarefa
Complete a implementação da função `sine_array()`:
1) Use a função `numpy.linspace` para criar um array `x` de `num` números de `start` até `stop`.
2) Crie um array `f` de números tal que $f = sin(x)$.
3) No bloco `main`, chame a função para criar um array `a` de $100$ números de $0$ até $3 * pi$
   e o array correspondente `b`.

Adicionamos algum código para plotar os dados resultantes: execute o script para ver o gráfico.

<div class="hint">
Utilize a função <a href="https://numpy.org/doc/stable/reference/generated/numpy.sin.html?highlight=sin#numpy.sin"><code>numpy.sin</code></a> para o array <code>f</code>.

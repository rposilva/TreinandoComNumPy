## Importar NumPy

Para acessar o NumPy e suas funções, importe-o em seu código Python assim:
```python
import numpy as np
```

O nome importado é frequentemente abreviado para `np` para melhorar a legibilidade do código. Esta é uma convenção amplamente adotada que você deve seguir para que qualquer pessoa trabalhando com seu código possa entendê-lo facilmente.

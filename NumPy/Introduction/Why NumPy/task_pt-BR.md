## Escolhendo o NumPy: os Benefícios

[NumPy](https://numpy.org/) é o pacote mais amplamente utilizado para computação científica em Python. Ele fornece um objeto de array multidimensional e diferentes objetos derivados (como arrays mascarados e matrizes), além de uma ampla variedade de maneiras de manipular arrays. Estas incluem manipulação matemática, lógica, forma, ordenação, seleção, E/S, transformadas de Fourier discretas, álgebra linear básica, operações estatísticas básicas, simulações aleatórias e muito mais. Arrays NumPy facilitam operações matemáticas avançadas e outros tipos de operações em grandes conjuntos de dados.

Aqui estão os principais benefícios de usar o NumPy:

- O NumPy ajuda a reduzir os loops e torna seu código mais claro e fácil de ler.
- O NumPy utiliza algoritmos escritos em C que são concluídos em nanossegundos em vez de segundos: toda a iteração explícita, indexação, etc., ocorre "nos bastidores" em código C otimizado e pré-compilado.
- Operações em grandes conjuntos de dados são executadas de forma mais eficiente e com menos código do que é possível com sequências incorporadas do Python. Menos linhas de código geralmente significam menos bugs.
- Uma vasta comunidade está constantemente trabalhando para manter o NumPy rápido, amigável ao usuário e livre de bugs.

Numerosos pacotes científicos e matemáticos baseados em Python utilizam arrays NumPy. Isso significa que, para trabalhar eficientemente com a maior parte do software científico/matemático baseado em Python de hoje, apenas saber como usar os tipos de sequência embutidos do Python já é insuficiente – é necessário também estar familiarizado com arrays NumPy.

O NumPy suporta totalmente uma abordagem orientada a objetos, começando com ndarray, o objeto no núcleo do pacote NumPy. Ndarray é uma classe que possui numerosos métodos e atributos. Muitos de seus métodos são espelhados por funções no espaço de nomes mais externo do NumPy, permitindo que o programador codifique no paradigma que preferir. Essa flexibilidade permitiu que o dialeto de arrays NumPy e a classe NumPy ndarray se tornassem a linguagem de facto para troca de dados multidimensionais usada em Python.

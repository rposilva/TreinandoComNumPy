## Visualização do Curso

<b>Visualização do Curso</b> mostra o cronograma do curso: uma lista de lições com tarefas.

<style>
img {
  display: block;
  margin-left: auto;
  margin-right: auto;
}
</style>
<img src="edu_course_view.png" class="center" width="75%"/>

Você pode navegar para qualquer tarefa dando um clique duplo no seu nome.

Para ocultar a janela de Visualização do Curso, clique no botão Janela de Ferramentas do Projeto ou pressione <span style="color: #509DD6">&shortcut:ActivateProjectToolWindow;</span>. Isso lhe dará mais espaço para as janelas do Editor e Descrição da Tarefa.

Para exibir a janela de Visualização do Curso oculta, clique novamente no botão Janela de Ferramentas do Projeto (ou pressione <span style="color: #509DD6">&shortcut:ActivateProjectToolWindow;</span>).

A forma como as imagens são representadas (como uma matriz 3D) pode resultar em uma grande quantidade de memória utilizada para produzir uma única imagem. Por exemplo, para salvar uma imagem de 100 × 100 em escala de cinza, 10.000 valores de pixels diferentes seriam armazenados. Uma maneira de permitir que os computadores salvem imagens sem ocupar grandes quantidades de memória (para aproximar os dados) é usar métodos da álgebra linear, como a [Decomposição em Valores Singulares](https://en.wikipedia.org/wiki/Singular_value_decomposition) (SVD). Isso permite comprimir o tamanho das matrizes de saturação enquanto retém os componentes mais importantes, ou seja, salvar uma imagem usando menos memória enquanto preserva a qualidade da imagem.

Aqui, usaremos o SVD para reconstruir uma imagem que usa menos informações de valores singulares do que a original, mas ainda retém muitas de suas características.

><i><b>Nota</b>: Usaremos o módulo [linalg](https://numpy.org/devdocs/reference/routines.linalg.html#module-numpy.linalg) do NumPy para realizar as operações neste tutorial. A maioria das funções de álgebra linear neste módulo também pode ser encontrada em [scipy.linalg](https://docs.scipy.org/doc/scipy/reference/linalg.html#module-scipy.linalg), e é geralmente incentivado a usar o módulo [scipy](https://docs.scipy.org/doc/scipy/reference/index.html#module-scipy) para aplicações no mundo real. No entanto, algumas funções no módulo `scipy.linalg`, como a função SVD, suportam apenas arrays 2D. Para mais informações sobre isso, consulte a [Referência sciopy.linalg](https://docs.scipy.org/doc/scipy/reference/tutorial/linalg.html). </i>

Para extrair informações de uma matriz dada, podemos usar o SVD para obter 3 arrays que podem ser multiplicados para obter a matriz original. Da teoria da álgebra linear, dada uma matriz real $m \times n$ $A$, o seguinte produto pode ser calculado:

$$U\Sigma V^T = A$$

onde $U$ é uma matriz unitária complexa $m \times m$, e $V$ é uma matriz unitária complexa $n \times n$. $\Sigma$ é uma matriz diagonal retangular $m \times n$ com números reais não negativos na diagonal, organizados do maior para o menor. Esses valores podem ser usados como um indicador da "importância" de algumas características representadas pela matriz $A$. $U$ e $V^T$ são matrizes [ortogonais](https://en.wikipedia.org/wiki/Orthogonal_matrix).
